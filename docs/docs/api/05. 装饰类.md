# 装饰类

装饰类的特点是将 html元素 转化为另一项 html元素。

大多数的装饰类只是在原有 html元素 的基础上，套一层具有特定css的外壳，以修改、增强原有的 html元素。所以叫做装饰类

## md

```js
id: "md",
name: "md",
process_param: ABConvert_IOEnum.text,
process_return: ABConvert_IOEnum.el,
```
